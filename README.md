# RestAssured_Java

This is a Java project that demonstrates API testing using RestAssured. RestAssured is a Java DSL for simplifying testing of REST based services built on top of HTTP Builder.

## Prerequisites

- Java 8 or higher
- Maven 3.x

## Setup

1. Clone the repository:
   ```
   git clone https://github.com/narkaromkar/RestAssured_Java.git
   cd RestAssured_Java
   ```

2. Install dependencies:
   ```
   mvn clean install
   ```

## Running Tests

To run the API tests:
```
mvn test
```

## Project Structure

- `src/main/java/` - Main source code
- `src/test/java/` - Test cases using RestAssured
- `pom.xml` - Maven configuration
